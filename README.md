# RICE 

Role: You have to create a deterministic calculator for a retail shopper

Intent: Should be able to 
* Add
* Subtract
* Multiply
* Divide
* Percentage
* Check History

Context: 
* Precision: Handle floating point arithemetics accurately, avoid python floating point issue 
* Display: Large, responsive output display with a "ghost" history preview of the previous calculations 
* Memory: Implementations of "Clear" and "All Clear" functionality 
* Interaction: Support for both mouse clicks and physical keyboard input 
* Performance: Low latency and responsive UI 
* Color: choose modern looking collor pallete

Enforcement: 
* Avoid making a "toy" calculator and follow the instructions carefully
