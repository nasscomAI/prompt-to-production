skills:

- name: retrieve_documents
  description: Loads all 3 policy text files individually and indexes their contents accurately by document name and section number.
  input: List of file paths to the policy text documents.
  output: A structured dictionary grouping each document's name to its parsed numbering and section text.
  error_handling: Return a clear error string indicating which document path failed.

- name: answer_question
  description: Searches the indexed documents and returns strictly a single-source answer with correct citation, or the precise refusal template.
  input: The user's question as a string, and the database dictionary generated by retrieve_documents.
  output: A string containing either the validated single-source answer (with section citation) or the exact refusal template string.
  error_handling: System instantly uses the designated refusal template if findings span multiple files (blending risk) or produce no distinct relevant matches.
