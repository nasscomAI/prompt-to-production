# skills.md — UC-X Policy Q&A Skills

skills:
  - name: retrieve_documents
    description: Loads the three specified policy documents and parses them into a structured index keyed exactly by document name and section number.
    input: File paths to the three policy text files (`policy_hr_leave.txt`, `policy_it_acceptable_use.txt`, `policy_finance_reimbursement.txt`).
    output: A structured index (e.g., a nested dictionary) mapping each document name and section number to its verbatim text content.
    error_handling: Throws a file loading error if any of the required documents are missing or unreadable.

  - name: answer_question
    description: Searches the indexed documents to find an explicit answer to a user's question, strictly enforcing single-source logic and mandatory citations.
    input: The user's natural language question and the structured index generated by `retrieve_documents`.
    output: A single-source factual answer securely paired with its exact document name and section citation.
    error_handling: Immediately halts and outputs the exact verbatim refusal template if the answer is not explicitly found, or if answering it requires prohibited cross-document blending.
