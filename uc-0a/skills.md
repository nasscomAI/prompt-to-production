# Skills & Tools

## 1. WebSearchTool
- **Function**: Searches the live web for real-time data.
- **Parameters**: query (string).

## 2. FileReadTool
- **Function**: Reads the content of local files for analysis.
- **Parameters**: file_path (string).

## 3. DataAnalysisTool
- **Function**: Uses Python (Pandas) to analyze CSV data.
