# UC-0A Agents

## 1. Classification Agent
- Reads complaint text
- Assigns category (Water, Road, Garbage, Electricity, Other)

## 2. Priority Agent
- Checks severity keywords
- Assigns priority (High, Medium, Low)

## 3. Validation Agent
- Checks for missing or empty complaints
- Flags invalid rows

## 4. Output Agent
- Formats output correctly
- Ensures CSV has required fields
