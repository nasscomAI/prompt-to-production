HIGH-FIDELITY CLAUSE SUMMARY:

Clause 2.1: Corporate devices must be used primarily for official work purposes.
Clause 2.3: Software installation requires written approval from the IT Department; only catalogued software is permitted.
Clause 2.6: Security agents must be active at all times; disabling them is a disciplinary offence.

Clause 7.2: Violations involving unauthorised access to restricted data will be reported to law enforcement.
