# UC-0B Agent Rules — Prevent Meaning Loss

- Every clause must be included in the summary
- No clause omission allowed

- Multi-condition clauses must preserve ALL conditions:
  - AND conditions must not be reduced
  - Time constraints must not be removed
  - Authority requirements must remain intact

- Do not soften obligations (must ≠ may)

- Do not introduce external knowledge

- If summarization risks meaning change:
  → Output clause verbatim
  → Mark as [VERBATIM]
