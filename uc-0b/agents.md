# UC-0B Summary Guardrails

## Meaning Preservation Rules
1. Every numbered clause must appear in the summary.
2. Preserve clause IDs in every bullet.
3. Never weaken obligation words:
   - must
   - requires
   - will
   - not permitted
4. Preserve all conditions:
   - deadlines
   - date windows
   - thresholds
   - approvers
   - exceptions
5. Multi-condition clauses must preserve ALL conditions.
6. For multi-approver clauses, list every approver explicitly.
7. If safe summarization is not possible:
   - quote verbatim
   - add REVIEW_REQUIRED
8. Final parity check:
   source clauses count == summary bullets count
