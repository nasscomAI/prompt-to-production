skills:
  - name: retrieve_policy
    description: Loads a .txt HR policy file and returns its content as structured, numbered sections.
    input: File path (string) of the .txt policy document (e.g., '../data/policy-documents/policy_hr_leave.txt').
    output: Structured representation (e.g., JSON or structured text) containing the policy clauses with their exact numbered references.
    error_handling: If the file is not found or cannot be read, returns an error message indicating the specific file issue. If the file lacks numbered sections, returns the raw text with a warning flag.

  - name: summarize_policy
    description: Takes structured HR policy sections and produces a compliant summary preserving all clauses and conditions.
    input: Structured sections (JSON or structured text) generated by `retrieve_policy`.
    output: A comprehensive text summary preserving every numbered clause, multi-condition obligations, and verbatim quotes for clauses that cannot be summarized without meaning loss.
    error_handling: If the input structure is invalid or missing clause numbers, returns an error requesting structured numbered sections. If it detects dropped conditions during processing, flags the specific clause and requests manual review.
