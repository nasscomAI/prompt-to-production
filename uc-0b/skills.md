skills:
  - name: retrieve_policy
    description: loads .txt policy file, returns content as structured numbered sections
    input: Filepath to a .txt policy document (e.g. policy_hr_leave.txt).
    output: Structured representation of verbatim text properly mapped directly to their 10 numbered sections/clauses.
    error_handling: Return descriptive error if the file cannot be accessed or lacks recognizable numbered sections.

  - name: summarize_policy
    description: takes structured sections, produces compliant summary with clause references
    input: Structured numbered sections generated by retrieve_policy.
    output: A compliant summary containing clause references for every extracted section. Must strictly preserve all binding verbs, multi-approver requirements, and constraints.
    error_handling: Flag and quote verbatim if a clause cannot be summarized without meaning loss, if you are unsure, or if it risks dropping a condition.
