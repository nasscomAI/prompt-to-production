skills:
  - name: retrieve_policy
    description: Loads a .txt policy file and returns its content as structured, numbered sections.
    input: File path of the policy document (.txt format).
    output: Structured list or dictionary of numbered clauses and their corresponding text.
    error_handling: Raises an error if the file is missing or unreadable; returns empty structure if no numbered sections are found.

  - name: summarize_policy
    description: Takes structured sections and produces a compliant summary with correct clause references, preserving all conditions.
    input: Structured numbered sections generated by retrieve_policy.
    output: A text summary representing every clause accurately without missing multi-condition requirements.
    error_handling: Quotes the clause verbatim and flags it if summarizing would cause meaning loss.
