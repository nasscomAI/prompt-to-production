skills:
  - name: retrieve_policy
    description: Loads a .txt policy file and returns its content as structured numbered sections.
    input: A string representing the file path to the .txt policy document.
    output: Structured numbered sections of the policy text (e.g., a dictionary mapping clause numbers to text).
    error_handling: If the file is missing or cannot be read, return an error indicating failure to load the policy file.

  - name: summarize_policy
    description: Takes structured sections and produces a compliant summary with clause references.
    input: Structured numbered sections generated by the retrieve_policy skill.
    output: A string containing the compliant summary with accurate clause references intact.
    error_handling: If a clause is ambiguous or cannot be summarized without meaning loss, quote it verbatim and flag it in the output.
