skills:

name: retrieve_policy
description: Loads the HR leave policy document and extracts structured numbered clauses.
input: File path to a .txt policy document containing numbered sections.
output: A structured list of tuples containing clause numbers and their text.
error_handling: If the file does not exist or contains no numbered clauses, return an error message and stop processing.

name: summarize_policy
description: Produces a clause-preserving summary of the policy while maintaining all obligations and conditions.
input: Structured clause list generated by retrieve_policy.
output: A text summary listing each clause number followed by its summarized content.
error_handling: If a clause cannot be summarized without losing meaning, return the clause verbatim and flag it.
