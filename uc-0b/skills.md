skills:
  - name: retrieve_policy
    description: Loads a .txt HR policy file and extracts its exact content into structured, numbered sections.
    input: File path to the .txt policy document.
    output: A structured List/Dict mapping each numbered clause directly to its verbatim text.
    error_handling: Return an error and explicitly refuse to process if the file is missing, illegible, or structurally broken such that clauses cannot be identified.

  - name: summarize_policy
    description: Processes structured policy sections into a compliant summary that preserves all exact multi-condition obligations without softening language or scope bleed.
    input: Structured numbered clauses generated by retrieve_policy.
    output: A precise text summary where every numbered clause is accounted for, using verbatim quotes where required to avoid meaning loss.
    error_handling: Halt execution and flag an error if standard policy practices must be inferred, external knowledge is required, or meaning cannot be safely represented.
