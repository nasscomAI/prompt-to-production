# skills.md

skills:
  - name: retrieve_policy
    description: Loads a .txt policy file and returns the parsed content as structured numbered sections.
    input: Path to the .txt policy file.
    output: A structured object or text format containing numbered sections matching the source document.
    error_handling: Return an error if the file cannot be read or if no numbered clauses can be detected in the text.

  - name: summarize_policy
    description: Takes structured sections and produces a compliant summary full of clause references without loss of meaning or conditions.
    input: Structured sections generated by retrieve_policy.
    output: A summarized text block with references to all input clauses.
    error_handling: Refuse to summarize if a clause's meaning would be lost; instead, quote the clause verbatim and flag it in the output.
