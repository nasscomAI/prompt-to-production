skills:
  - name: retrieve_policy
    description: loads .txt policy file, returns content as structured numbered sections
    input: File path to the .txt policy document (string)
    output: Internal structure containing the document content parsed into numbered sections
    error_handling: Return an error if the file does not exist, cannot be read, or contains no parseable sections

  - name: summarize_policy
    description: takes structured sections, produces compliant summary with clause references
    input: Structured numbered sections generated by retrieve_policy
    output: A text summary string that complies with all enforcement rules, mapping each summary point to its original clause reference
    error_handling: Return an error if the input sections are malformed or missing key obligation keywords
